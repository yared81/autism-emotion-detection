## 🎯 Project Overview

The Autism Emotion Detection project is a web-based application that uses deep learning to detect and classify human emotions from facial expressions in real-time. The system supports both webcam live detection and image upload functionality.

### Key Features
- Real-time emotion detection via webcam
- Image upload for emotion analysis
- Custom-trained CNN models
- Modern, responsive web interface
- RESTful API backend
- Comprehensive model training tools

### Technology Stack
- **Frontend**: React, Tailwind CSS
- **Backend**: Flask (Python)
- **ML Framework**: TensorFlow/Keras
- **Computer Vision**: OpenCV
- **Model**: Custom CNN or DeepFace
